# RecipeRecommendor
 A simple web application which recommends recipies on the basis of the ingredients you provide as imput
 This works on simple vectorization method which predicts the data from the trained model names "recipe_model.pkl" and gives the recommended dishes which can be made.
## Working
The working here is html for fromtend and flask for the backend
### Why flask ?
Flask is a light weight python backend framework ideal for developing simple projects
